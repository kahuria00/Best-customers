**Installations needed**
1. Python from https://www.python.org/
    		

2. Pandas Library
```pip install pandas```


Running instructions

I. Clone the repository
II. Navigate to folder via cmd/control
III. write ```python debtor.py``` to run the executable file.
IV. you will be propmted to enter csv name ```Enter csv name: ```
V. enter one file name at a time e.g ```transaction_data_1.csv```
VI. you will be prompted to enter number of paying customers you want to retrieve ```"Enter number of paying customers you need: "```
VII. Your result will be shown
